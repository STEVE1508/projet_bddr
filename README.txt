Membres du groupe : steve LILEKE, lindor FALL et hugo MASSON
Notre projet consiste à créer une base de données multi-tables. Nous les croiserons ensuite pour permettre à l'utilisateur d'évaluer le niveau de risque de son trajet en se basant sur des caractéristiques telles que : le type et le nombre de moteurs propulseurs de l'appareil,les conditions météo,le modèle ou constructeur de l'appareil ou encore le motif de déplacement. Nous pourrons dans la mesure du possible lui proposer de reporter son voyage ou de changer de compagne d'aviation.
On a un dossier mp (lien du dossier) dans lequel :
- un fichier mp/monappli/models.py qui décrit la structure de notre base de données
- un fichier mp/remplissage.py avec lequel on a peuplé nos tables
- un fichier mp/trials.py avec lequel nous avons fait quelques essais de requêtes en intérrogeant nos tables dans le shell django
