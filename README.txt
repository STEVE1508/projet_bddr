Membres du groupe : steve LILEKE, lindor FALL et hugo MASSON
Notre projet consiste à créer une base de données multi-tables.
Avec le framework django , nous concevrons une application web interagissant avec ladite base de données pour permettre à l'utilisateur d'évaluer le niveau de risque de son trajet en se basant sur des caractéristiques telles que : 
le type et le nombre de moteurs propulseurs de l'appareil,
les conditions météo,
le modèle ou constructeur de l'appareil ou encore le motif de déplacement. 
Nous pourrons dans la mesure du possible lui proposer de reporter son voyage ou de changer de compagnie d'aviation.
L'intégralite du projet se trouve dans le dossier improve.7z
Pour lancer l'application , il suffit de téléchager dossier improve.7z et le décompresser. Il faudra ensuite se mettre dans ce repertoire et exécuter le fichier lancement.py qui se trouve directement dans le répertoire principal. Il est nécessaire de lancer l'application en exécutant lancement.py pour que les liens à l'intérieur de l'application fonctionnent
